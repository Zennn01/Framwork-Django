# Frontend Prototype

Prototype ini adalah halaman frontend murni untuk UI travel marketplace. Belum ada integrasi Django, database, authentication, atau upload media.

## Cara Menjalankan

Buka `frontend/index.html` langsung di browser.

## Struktur

- `assets/css/`: CSS modular untuk reset, variable, layout, komponen, dan responsive.
- `assets/js/`: JavaScript vanilla untuk slider, kategori, dan animasi.
- `assets/images/banners/`: gambar destinasi dan hero.
- `assets/images/hotels/`: gambar hotel.

## Catatan

Hero carousel menampilkan 4 foto di desktop dan menyesuaikan menjadi 3, 2, atau 1 foto pada layar yang lebih kecil.
